### 二,ExOsip2
 1. ####  介绍
|库名  | 功能 |
|--|--|
| Osip2 | 一个开源的SIP协议栈，使用C编写，主要提供解析SIP和SDP消息的API和事物处理的状态机 |
 | Exosip2| 是对Osip2协议栈的封装和调用，作为Osip2的一个扩展协议集，使得Osip2更容易被使用|
